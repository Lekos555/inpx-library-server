# Changelog

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [1.7.8] - 2026-06-19

### Changed

- Меню команд Telegram-бота упорядочено по секциям (поиск → личное → аккаунт).

### Fixed

- **Самопривязка Telegram** — `/start link_…` обрабатывается до проверки доступа; привязка работает в режимах `linked_only` и с whitelist.
- **Админка** — Telegram ID сохраняется только у выбранного пользователя (автозаполнение браузера, поле `accountUsername`).
- Дубликаты `telegram_id` в БД очищаются при старте; при привязке ID снимается с других аккаунтов.

[1.7.8]: https://github.com/Habsaec/inpx-library-server/releases/tag/v1.7.8

## [1.7.7] - 2026-06-19

### Added

- **Telegram: привязка аккаунта** — самопривязка в профиле (`/profile/settings`) через deep link; отвязка из профиля или `/unlink` в боте; ручная привязка Telegram ID в админке.
- **Telegram: персональный контент** — после привязки: `/shelves`, `/favorites`, `/recommended`, `/me`.
- **Telegram: per-user доступ** — галочка «Доступ к Telegram-боту» у каждого пользователя; режимы доступа в `/admin/telegram`.
- **Email: per-user доступ** — галочка «Отправка книг на email»; скрытие кнопок и блокировка API при запрете.
- **Админка → Пользователи** — отображение и редактирование e-reader email; метки «TG запрещён» / «Email запрещён».
- **Главная** — welcome-баннер с цитатами (ru/en), hero-карточка для «Продолжить чтение», анимация появления блоков.

### Changed

- Рекомендации в Telegram-боте: до 5 книг на страницу, максимум 10 (вместо полного кеша сайта).
- Аннотации на главной подгружаются из кеша метаданных, если их нет в списке книг.

### Fixed

- Кнопка «Привязать Telegram» больше не зависает со спиннером (GET-ссылка + отложенный disable на POST-формах).
- Галочки доступа TG/email в админке корректно сохраняются при повторном включении.

[1.7.7]: https://github.com/Habsaec/inpx-library-server/releases/tag/v1.7.7

## [1.7.6] - 2026-06-14

### Added

- **Telegram-бот** — поиск книг, авторов и серий; умный ввод без команд; скачивание в нескольких форматах.
- Карточки книг в боте с обложкой и аннотацией.
- Страница **Админка → Telegram-бот**: токен, список пользователей, описание профиля, текст `/start`.
- Режим **webhook** (`TELEGRAM_WEBHOOK_URL`) или long polling; меню команд; кнопка «Назад»; очистка старых сообщений при пагинации.
- Безопасные URL **`/book/b64/…`** для книг с управляющими символами в ID (обход 400 от nginx и прокси).
- Профиль: блоки «Читаю», закладки читалки и заметки; навигация по разделам аккаунта.
- Пагинация в избранном (книги и прочитанные); раздел «Прочитанные».
- Читалка: выбор шрифта (системные и Google Fonts), пресеты вёрстки, настройки полей и ширины колонки.
- Улучшения **TTS**: чтение в фоне, остановка долгим нажатием.

### Changed

- «Продолжить чтение» на главной подгружается отдельно и не блокирует страницу.
- При сортировке по серии номер книги в серии учитывается раньше релевантности поиска.
- OPDS и веб-ссылки на книги используют схему book-ref.

### Fixed

- Удаление записи из истории чтения возвращает корректные ошибки при неверном ID или отсутствии записи.
- Настройки Telegram-бота в админке сохраняются надёжно (флаг `telegram_configured`, без перезаписи env).
