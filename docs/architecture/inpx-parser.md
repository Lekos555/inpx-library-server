Порядок и интерпретация полей в .inp(14 полей)
AUTHOR — Авторы. Формат: Фамилия,Имя,Отчество. Разделитель нескольких авторов — :
GENRE — Коды жанров. Разделитель — :
TITLE — Название книги.
SERIES — Названия серий. При наличии нескольких серий разделитель — :
SER_NO — Номера в сериях. Разделитель — :. Соответствуют порядку в поле 4
FILE — Имя файла книги внутри ZIP-контейнера (без расширения)
SIZE — Размер файла в байтах (Integer)
LIB_ID — Уникальный ID книги в системе библиотеки
DEL — Статус удаления (0 — активна, 1 — удалена/скрыта)
EXT — Расширение файла книги (fb2, epub, mobi)
DATE — Дата регистрации (YYYY-MM-DD)
LANG — Язык книги (ru, en и т.д.)
LIBRATING — рейтинг книги
KEYWORDS — Теги или ключевые слова
